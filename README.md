# snake_and_priest

## Quick start
```cli
$ pip install -r requirements.txt
$ python3 snake_and_priest.py < examples/input-0.txt
```

### Building C program
```cli
$ ./run.sh solution
$ ./solution examples/input-0.txt
```

### Building animation
```cli
$ sudo apt install libsfml-dev
$ ./run.sh game
$ ./game-sfml examples/input-0.txt
```
